# Fonts

`NotoSansSC-Regular.otf` and `NotoSansSC-Bold.otf` are used by the dynamic Open Graph image renderer so Chinese titles can render from local fonts during build and deploy.

The expected font family is Noto Sans SC, distributed by Google under the SIL Open Font License. If you replace the files, keep the same filenames or update `lib/og-image.tsx`.
